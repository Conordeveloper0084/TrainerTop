# TrainerTop

O'zbekistonning #1 fitness platformasi. Trenerlar profil ochadi, darslik yaratadi, shogirdlar bilan ishlaydi. Foydalanuvchilar ishonchli trener topadi, darslik sotib oladi, chat orqali muloqot qiladi.

## Texnologiyalar

- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage
- **Auth:** Supabase Auth
- **AI:** OpenAI GPT-4o-mini
- **Real-time:** Supabase Realtime

## O'rnatish

```bash
git clone https://github.com/Conordeveloper0084/TrainerTop.git
cd TrainerTop
npm install
cp .env.example .env.local
# .env.local ga real kalitlarni kiriting
npm run dev
```

## Deploy

Vercel'ga deploy qilish:
1. GitHub'ga push qiling
2. vercel.com da import qiling
3. Environment variables sozlang
4. Deploy
